# Movie Data Analysis (Python)

This project performs data analysis on a movie dataset using Python.

## Features
- Handling missing values
- Removing duplicates
- Sorting by release year
- Finding highest & lowest ratings
- Genre-based analysis
- Adding rating score column

## Tools Used
- Python
- Pandas
- Google Colab
